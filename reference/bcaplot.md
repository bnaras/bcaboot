# BCa plot (deprecated)

**\[deprecated\]**

`bcaplot()` was deprecated in bcaboot 1.0 in favour of
[`autoplot()`](https://ggplot2.tidyverse.org/reference/autoplot.html).

## Usage

``` r
bcaplot(vl, ...)
```

## Arguments

- vl:

  Output of a bcaboot computation function.

- ...:

  Additional arguments (ignored).
